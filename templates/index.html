Algerian Forest Fire Prediction

Leveraging Machine Learning to Forecast Fire Weather Index (FWI)

Project Overview

The Goal

This project is dedicated to predicting the Fire Weather Index (FWI), a critical measure of fire danger, using various meteorological factors from Algerian forest regions. Accurate FWI forecasts are essential for timely fire prevention and resource deployment, saving lives and natural resources.

Data & Features

The model utilizes a cleaned dataset of daily weather observations, focusing on critical fire codes (FFMC, DMC, and ISI) alongside meteorological inputs like Temperature, Relative Humidity (RH), Wind Speed (Ws), and Rainfall (Rain). Feature engineering was performed to select the most impactful variables for prediction.

Model Performance & Selection

Our machine learning pipeline evaluated several regression models, ultimately selecting the one that delivered the highest predictive capability and stability.

  • ✓
    Primary Model: Ridge Regression (Regularized Linear Model)
  • ✓
    Key Metric (R² Score): Achieved a robust score of ~0.983 on the test set, demonstrating excellent fit and generalization capability.
  • ✓
    Preprocessing: All features are scaled using Standard Scaling to ensure the model trains efficiently and accurately.
